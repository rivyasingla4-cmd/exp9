# AIM
To study the basics of NumPy arrays and perform different operations such as checking dimensions, shape, datatype, generating arrays using built-in functions, and performing mathematical operations on arrays.

# THEORY
NumPy (Numerical Python) is a Python library used for numerical and scientific computing. It provides powerful N-dimensional array objects and many built-in mathematical functions to efficiently perform operations on arrays.

Important Commands Used

import numpy as np
Imports the NumPy library and assigns it the alias np for easier usage.

np.array()
Creates a NumPy array from a Python list or nested lists.

ndim
Returns the number of dimensions of an array.

shape
Returns the size of each dimension of the array in tuple form.

dtype
Displays the data type of elements stored in the array.

np.zeros((rows, columns))
Creates an array filled with zeros of the specified shape.

np.eye(n)
Generates an identity matrix of size n × n.

np.arange(start, stop, step)
Generates an array with values in a specified range with a step interval.

np.linspace(start, stop, num)
Generates evenly spaced numbers between start and stop.

Array Arithmetic Operations (+, *)
Performs element-wise mathematical operations on arrays.

np.mean()
Calculates the average (mean) value of array elements.

np.median()
Returns the median (middle value) of the dataset.

np.max()
Returns the maximum value in the array.

np.min()
Returns the minimum value in the array.

np.sum()
Calculates the sum of all elements in the array.
